# NTHO. E-Commerce

A modern, minimalist e-commerce website for luxury t-shirts, built with Next.js 16 and Tailwind CSS.

## Features

- **Product Catalog**: Filter by category and search by name.
- **Shopping Cart**: Real-time cart management with local storage persistence.
- **Checkout**: Integrated Paystack payment gateway.
- **Responsive Design**: Mobile-first approach with a premium look and feel.
- **Modern UI**: Clean animations, backdrop blurs, and sophisticated typography.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **State Management**: React Context API
- **Icons**: Lucide React
- **Payment**: React Paystack

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To build the application for production:

```bash
npm run build
```

## Note

This project uses mock data for products and a test public key for Paystack. In a real environment, replace these with actual API calls and valid credentials.
